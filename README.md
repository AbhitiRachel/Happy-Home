# HAPPY HOME



## Overview

HAPPY HOME is an application developed completely for donors to donate anything to a welfare home. This platform is designed to avoid scammers who intend to take donated items for their own sake. The system records all donor information, including the date and time items are delivered and received in the database. It allows users to choose the category of their donation and provides lists of registered welfare homes.

## Objectives

* To ease donors who wish to make a donation but do not know where to donate it.


* To avoid fraud issues regarding stolen money or any donation item.


* To raise donations for abandoned and lonely senior citizens.


* To make a user-friendly system for volunteers.


* To provide emotional support to overcome social isolation.



## Features and Functionality

* **User Accounts:** Secure sign-up and login features for donors, along with a profile dashboard.


* **Payment Portal:** Includes a secure payment gateway for fund donations, complete with receipts and a thank you message.


* **Welfare Donations:** Allows users to schedule food and clothes donations by selecting specific dates, time slots, and NGOs.


* **Location Services:** Provides contact numbers and locations of old age homes for visitors.


* **Feedback System:** Includes a feedback and rating system allowing donors to rate welfare homes.


* **Admin Dashboard:** A dedicated portal for administrators to track funds donated, clothes donated, and organized events.



## Software Requirements

* **Tkinter:** A Python library used for the development of the graphical user interface (GUI).


* **MySQL:** An open-source relational database management system used to store user and donation data.


* **PyCharm IDE:** The integrated development environment used to build the application.
